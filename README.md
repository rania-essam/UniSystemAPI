# Simple UniAPI

Simple UniAPI is a basic university management REST API project.

It provides simple APIs for managing:

* Students
* Courses
* Departments

The project is designed for learning and practicing backend development concepts such as:

* RESTful APIs
* CRUD operations
* Routing
* Database models
* JSON handling
* Pagination
* DTOs
* Automapper
* Swagger/Postman testing 

This API can be extended later with authentication, validation, and additional university features.
